## 上传命令模板

```bash
# 使用 curl 上传到 Supabase Storage
curl -X POST "https://pahyccjqmlkyvbtwjydg.supabase.co/storage/v1/object/media/文件名" \
  -H "Authorization: Bearer YOUR_SERVICE_ROLE_KEY" \
  -H "Content-Type: application/pdf" \
  --data-binary @"本地文件路径"
```
